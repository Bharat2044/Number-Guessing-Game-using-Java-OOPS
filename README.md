## How to Run:

### Clone the Repository
```sh
git clone https://github.com/Bharat2044/Number-Guessing-Game-using-Java.git
```

```sh
cd Number-Guessing-Game-using-Java
```

### Compile the Code
```sh
javac GameTest.java
```

### Run the Game
```sh
java GameTest
```
